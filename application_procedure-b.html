Instructions for On-Line Application ::

Read the following instructions carefully before submission of On-line Application.

For Admission to First Year B.A./B.Sc. (Hons./General/Major), B.B.A.(Hons.), B.Voc. Programme, Diploma and Advanced Diploma Courses.

The Steps to be taken for On-Line submission of application:

STEP 1: Fill all Fields in On-Line Application Form sequentially

(use Google Chrome or Mozila Firefox for better view)

STEP 2: Use separate On-line Application Form for each course.

STEP 3: Record unique Application ID No which will be generated automatically after submission of the On-Line Application Form for future reference.

STEP 4: Print Bank Challan (Student and Bank copy) and Application Form (Student's Copy) after submission of On-Line Application.

(* in any case, if you fail to print Bank Challan / Application Form (Student's Copy) then go to the top of this page and Click 'Print Application Form / Bank Challan' menu just right side of 'Apply Online' menu)

STEP 5: Deposit Only Cash of Rs. 200.00 + Bank Charge for each On-Line Application using Only this Printed Bank Challan at any branch of the following Banks:

 BANK NAME 
State Bank of India
ICICI Bank

* Nothing is required to send to the college, please wait for Merit List that will be published in this website.


For merit list and subsequent admission follow the important dates in the website.
